Guide on how to install and use quicklauncher:

1. First of all you need to have this version of python installed in your pc (remember to install it in C:\ drive only)
https://www.python.org/ftp/python/2.7.11/python-2.7.11.msi
2. Now extract both the files from winrar archive in your Toontown Rewritten directory.
3. Open TTRQuickLauncher.py with notepad
4. Now press CTRL + G and enter 22 in the text field and press enter.
5. Edit the username and password and then save the file.
6. Now open PlayTTR.bat, it will log you in TTR automatically (skipping the login and update screen)
